Database Used: MySQL Workbench
Framework: NodeJS and React

Set Up
1. Just install the dependencies on the backend and client folder by running 'npm i' command on both folders
2. To change database credentials and admin credentials, use the .env file located on ./backend/.env

Running the program
1. You need 2 terminals open
2. Go the backend folder and run the backend with the command 'node server.js' and make sure that the mysql local server is running so that the backend can connect to the server
3. Go the client folder and launch the front end using 'npm start'
